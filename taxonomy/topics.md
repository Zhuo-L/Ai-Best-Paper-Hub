# 主题标签体系（topics.md）

> 说明：Topics 用于跨会议检索与归档。建议每条论文选 1–3 个主标签 + 0–2 个辅标签。  
> 维护规则：新增标签前先检查是否已有同义标签；尽量用英文 snake_case；必要时在“同义词”中登记。

## 主标签（Core）
- foundation_models（大模型/预训练）
- llm_reasoning（推理与规划）
- multimodal（多模态）
- representation_learning（表征学习）
- optimization（优化理论与算法）
- probabilistic_ml（概率/贝叶斯）
- causal_inference（因果推断）
- reinforcement_learning（强化学习）
- imitation_learning（模仿学习）
- online_learning（在线/持续学习）
- continual_learning（持续学习）
- meta_learning（元学习）
- graph_learning（图学习/GNN/传播）
- contrastive_learning（对比学习）
- generative_models（生成模型）
- diffusion_models（扩散模型）
- vae_gan（VAE/GAN 等）
- efficient_ml（高效/压缩/加速）
- distributed_ml（分布式/系统）
- robustness_security（鲁棒/安全）
- privacy_fairness（隐私/公平）
- interpretability（可解释性）
- uncertainty（不确定性估计）
- dataset_benchmark（数据集与评测）

- 3d_vision（3D/点云/重建）

- medical_imaging（医学影像）

## NLP 方向（Language）
- nlp_pretraining（预训练与微调）

- machine_translation（机器翻译）
- summarization（摘要）
- qa_dialogue（问答/对话）
- nlp_evaluation（评测与基准）

## 同义词与规范（Examples）
- “GNN / message passing / label propagation” → graph_learning
- “FM / LLM / foundation model” → foundation_models
- “V&L / VLM” → vision_language 或 multimodal（按侧重点选）
